=== Риски

Инфраструктурный риск

[cols="3,^2,^8"]
|===
|Риск |Влияние | Описание

|утечка данных пользователей
|Репутационные, финансовые
|Личная инфомация такая как номер телефона, день рождения, реальные ФИО, история переписки

|Зависимость от сторонних продуктов
|Потеря части функциональности при выходе зависимостей из строя
|Например потеря аватаров если используется сервис типа gravatar

|Поддержка мультиплатформенности
|Разный интерфейс - несколько команд разработки, сложность обеспечения одновременного релиза на разных платформах
|Поведение приложения на различных мобильных платформах различно, веб версия не поддерживает то что поддердивает десктопная и т.д.

|Проблемы масштабируемости
|Система не справляется с нагрузкой - пользователи начинают обсуждать рабочие вопросы в непредусмотренных для этого средствах
|Пиковая нагрузка вызывает существенные лаги, потерю возможности пересылки мелия и т.д.

|Нарушение законодательства пользователями
|Возможность уголовного или вдминистративного наказания
|Пользоватеи публикуют противоправный контент

|Проблемы безопасности
|Возможность приникновения порсторонних лиц в корпоративный мессенждер, доступ к коммерческой тайне
|Доступ через уязвимости или утекшие пароли

|===
