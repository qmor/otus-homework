=== Внешние интеграции

Внешней интеграцией для системы является LDAP для получения сведений о сотрудниках (в случае self-hosted) или внешний источник данных о пользователях в случае облачного размещения решения

Также имеется интеграция с email сервером предприятия для авторизованной рассылки польщователям OTP для входа
